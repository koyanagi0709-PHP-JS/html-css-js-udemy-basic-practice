### スタイル ルール
1. プロトコルは、HTTPSｗｐ使う。
    + SSH化していない限り、ソースの参照をする場合は、HTTPSプロトコルを使用します。
2. インデントは2つの半角スペースを使用する
    + タブを使用したり、タブやスペースを混在したりしないようにします。
3. 文字は半角小文字のみを使用する。
    + 適用されるのは、要素名、属性名、属性値（text/CDATA以外）となります。
4. 末尾にある空白は削除する。
5. エンコーディング（文字コード）は、utf-8（バイトオーダーマークなし）を使用する
6. 必要な箇所にはコメントをつけ、コードの説明を入れる。
    + ただし、オプション扱いになるので、必須ではない。
7. コメントでToDoを記述する場合のルール
    + ToDoの横にユーザー名またはメーリングリストを記述し、実現することアイテム名などを記述します。
